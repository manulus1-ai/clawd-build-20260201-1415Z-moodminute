# MoodMinute

**Problem:** People want mood tracking without writing long journals.

**Source:** https://developer.mozilla.org/en-US/docs/Web/API/Periodic_Background_Synchronization_API

---

## Cycle 1 — Product Manager
- MVP: 10-second check-in (emoji + tag) + streak + weekly insights.
- Core loop: check-in → insight → optional share of “week arc”.
- North star: check-ins/week; D30 retention.
- Safety: avoid diagnostic claims.

## Cycle 2 — Marketing Genie
- Hook: “10 seconds a day.”
- Share moment: weekly arc link (opt-in).
- Growth: friend accountability circles.
- Distribution: wellness creators.

## Cycle 3 — Designer
- Calm pastel; big emoji selector; minimal text.
- Gentle animations; reduce motion toggle.
- Privacy emphasis.
- Accessible contrast option.

## Cycle 4 — Software Developer
- Local-first; offline; optional background sync where supported.
- Export/import JSON; share state permalinks.
- Optional: notification reminders.
